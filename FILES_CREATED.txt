INTEGRATION POINTS
================================================================================

How to integrate into existing codebase:

1. Obstacle Detection Integration:
   robot.obstacle_detector = ObstacleDetector(expected_costmap)

   Automatic in update() cycle when robot.state == NAVIGATING

2. DWA Planning Integration:
   robot.dwa_planner = DWAPlanner(...)
   velocity, metrics = robot.dwa_planner.plan(...)

3. Alert System Integration:
   robot.alert_generator = AlertGenerator()
   hub = AlertHub()
   decision = hub.process_alert(alert)

4. Dashboard Integration:
   dashboard.update_alerts(alerts)
   dashboard.add_obstacle(position, radius)

5. Navigation Visualization:
   animator = NavigationAnimator(factory_env, path, headings)
   fig = animator.create_grid_visualization()
